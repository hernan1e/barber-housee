# CHANGELOG - THE BARBER HOUSE Website Updates

## Overview
All references to "Quilamapu" have been removed and the website has been updated with improved styling and color consistency.

## Changes Made

### 1. Removed All Quilamapu References
- Updated page titles, headings, and content text
- Removed "Quilamapu" from social media handles and references
- Updated location references from "Temuco, Araucanía, Chile" where appropriate
- Cleaned up sucursal/branch names and locations

### 2. Improved Logo Integration
- Added white background with padding to logo in navigation bars for better visibility
- Enhanced logo presentation in footer with consistent styling
- Added prominent logo display in hero section of homepage
- Removed problematic mix-blend-mode that was causing visibility issues

### 3. Color Palette Adjustments
- Changed overall site background from white/gray-50 to dark (#0a0a0a) for consistent premium look
- Updated service cards from white to dark gray (#2a2a2a) with subtle borders
- Improved text contrast with light gray text (#300 series) on dark backgrounds
- Maintained gold accent color (#d4af37) for branding consistency
- Updated form elements to match dark theme with proper contrast

### 4. Image Quality Improvements
- Added auto=format&fit=crop parameters to Unsplash image URLs for better quality
- Maintained premium barber shop imagery that matches brand identity

### 5. Consistent Styling Across Pages
- Applied consistent dark theme to all pages (index, servicios, resenas, reservas)
- Updated navigation menus with dark backgrounds and proper hover states
- Improved form elements with dark theme styling
- Enhanced visual hierarchy with proper contrast and spacing

- blosoom/index.html

## Result
The website now presents a cohesive, premium dark-themed barber shop experience with:
- Zero Quilamapu references
- Properly integrated logo with enhanced visibility
- Consistent color palette aligned with brand identity
- Professional imagery throughout
- Improved user experience with better contrast and visual hierarchy
